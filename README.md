# Reader Lite 一键阅读模式

一个轻量的 Chrome / Edge 浏览器扩展，用于从当前网页中提取主要正文内容，并切换到更干净、接近阅读器的阅读界面。

## 功能特点

- 一键切换当前网页到阅读模式
- 自动识别文章主体区域，优先处理 `article`、`main` 和主要内容容器
- 尽量保留正文中的标题、段落、列表、表格、代码块、图片等内容
- 自动移除导航、侧边栏、广告、评论等常见干扰元素
- 支持调节字号、行距、阅读宽度和正文对齐位置
- 支持调整图片显示比例，或直接隐藏图片
- 阅读设置会通过本地存储自动记忆
- 扩展徽标会显示当前页面是否处于阅读模式
- 支持按 `Esc` 或再次点击扩展图标退出阅读模式

## 工作方式

扩展会在进入阅读模式时评估页面中可能的正文容器，克隆并清理主体内容，然后临时替换为一个简洁的阅读视图。退出时会恢复原始页面 DOM、样式表和滚动位置。

## 当前限制

- 只对普通 `http` / `https` 页面生效
- 对正文极短、结构异常或高度动态的网站，正文识别结果可能不够理想
- 浏览器内部页面、扩展页和应用商店页面通常无法使用

## 安装方法

1. 下载或克隆本项目到本地
2. 打开浏览器并进入 `chrome://extensions/`
3. 打开右上角的“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择当前目录中的 `reader-lite` 文件夹

## 使用方法

1. 打开任意文章页或内容页
2. 点击浏览器工具栏中的 Reader Lite 图标
3. 页面会切换到简洁阅读视图
4. 将鼠标移动到页面底部附近，调出浮动工具栏
5. 按需调节字号、行距、宽度、位置和图片显示方式
6. 按 `Esc` 或再次点击扩展图标即可退出阅读模式

## 工具栏选项

- `字号`：增大或减小正文文字大小
- `行距`：在紧凑、中等、宽松三档之间切换
- `宽度`：选择窄、中、宽三种阅读宽度
- `位置`：选择左、中、右三种正文排版位置
- `图片`：缩放图片显示比例，或直接隐藏图片

## 项目结构

- `manifest.json`：扩展配置与权限声明
- `background.js`：处理图标点击、徽标状态同步和阅读模式开关
- `content.js`：正文识别、内容清理、阅读视图生成、工具栏交互和页面恢复逻辑
- `content.css`：阅读模式样式和浮动工具栏样式
- `icons/`：扩展图标资源

## 技术栈

- Manifest V3
- Vanilla JavaScript
- Chrome Extension APIs
- `chrome.storage.local` 本地配置持久化

## 适用场景

- 阅读博客、教程、技术文档和长文章
- 屏蔽网页边栏、广告和冗余模块
- 在不同屏幕尺寸下获得更稳定的正文阅读体验
